Lipi Lekhika supports all the **Modern Indian Scripts** including Devangari, Bengali, Kannada, Gujrati, Telugu, Tamil, Malayalam, Odia, Assamese, Gurumukhi and Sinhala. There are also two special output targets _Normal_ and _Romanized_.

## Supported Scripts

- Devangari
- Bengali
- Kannada
