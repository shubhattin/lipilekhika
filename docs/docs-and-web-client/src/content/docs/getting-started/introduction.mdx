Lipi Lekhika is a transliteration tool for Indian [Brahmic Scripts](https://en.wikipedia.org/wiki/Brahmic_scripts).
Supports all the **Modern Indian Scripts** including Devangari, Bengali, Kannada, Gujarati, Telugu, Tamil, Malayalam, Odia, Assamese, Gurumukhi and Sinhala. There are also two special output targets _Normal_ and _Romanized_.
[Full List of Supported Scripts](/reference/supported_scripts)

**Romanized** is a standard based on [IAST](https://en.wikipedia.org/wiki/International_Alphabet_of_Sanskrit_Transliteration) and [ISO 15919](https://en.wikipedia.org/wiki/ISO_15919). **Normal** is an opinionated standard inspired by [ITRANS](https://en.wikipedia.org/wiki/ITRANS) and [Harvard-Kyoto](https://en.wikipedia.org/wiki/Harvard-Kyoto).

## Transliteration

You can convert text from one script to another with a simple interface customizable interface.
Support is also available for [Custom Transliteration Options](/reference/custom_trans_options).

## Typing Tool

Lipi Lekhika comes a simple, fast and accurate typing tool for Indian Languages. It enables and empowers you to type in Indian Languages with **full speed and accuracy**.
It employs a predictable and intuitive approach for mapping keys in Indian Scripts.

## Multiple Programming Language Support

Lipi Lekhika package is available for [JavaScript](https://www.npmjs.com/package/lipilekhika), [Python](https://pypi.org/project/lipilekhika/), [Go](https://pkg.go.dev/github.com/shubhattin/lipilekhika/packages/go/lipilekhika) and [Rust](https://crates.io/crates/lipilekhika).
These Packages support Script Transliteration and offer utilities for implementing a typing tool based on the core transliteration engine.
