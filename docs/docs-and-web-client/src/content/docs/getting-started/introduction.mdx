Lipi Lekhika is a transliteration tool for Indian [Brahmic Scripts](https://en.wikipedia.org/wiki/Brahmic_scripts).
Supports all the **Modern Indian Scripts** including Devangari, Bengali, Kannada, Gujrati, Telugu, Tamil, Malayalam, Odia, Assamese, Gurumukhi and Sinhala. There are also two special output targets _Normal_ and _Romanized_.

**Romanized** is a standard based on [IAST](https://en.wikipedia.org/wiki/International_Alphabet_of_Sanskrit_Transliteration) and [ISO 15919](https://en.wikipedia.org/wiki/ISO_15919). **Normal** is an opinionated standard inspired by [ITRANS](https://en.wikipedia.org/wiki/ITRANS) and [Harvard-Kyoto](https://en.wikipedia.org/wiki/Harvard-Kyoto).

## Typing Tool

Lipi Lekhika comes a simple, fast and accurate typing tool for Indian Languages. It enables and empowers you to type in Indian Languages with **full speed and accuracy**. It employs a predictable and intuitive approach for mapping keys in Indian Scripts.

## Vedic Sanskrit

Almost all modern Indian Scripts support Vedic Sanskrit Accent Symbols. Lipi Lekhika comes with robust **typing** and **trasnliteration** support for Vedic Sanskrit.
