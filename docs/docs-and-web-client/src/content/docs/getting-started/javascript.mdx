Lipi Lekhika offers a typesafe, tree shakable package for JavaScript/TypeScript. It has support for all major javascript packaging formats namely **ESM**, **CommonJS** and **UMD**.

## Usage

For transliteration use the `transliterate` function. It returns a `Promise<string>`.

```ts
import { transliterate } from 'lipilekhika';

const result = await transliterate('na jAyatE mriyatE vA', 'Normal', 'Devanagari');
console.log(result); // न जायते म्रियते वा
```

- For commonjs you can use the `require('lipilekhika')` syntax.
- The Script/Language names are type safe so you will get editor support for autocompletion for names(including aliases).
- You can also pass in **custom transliteration options** to the `transliterate` as the fourth argument to the function.

## Using Without a Package Manager

You can also the package directly in the browser using the ESM and UMD Packages

### ESM in Browser

```html
<script type="module">
  import { transliterate } from 'https://cdn.jsdelivr.net/npm/lipilekhika/dist/esm/index.mjs';
  transliterate('na jAyatE mriyatE vA', 'Normal', 'Devanagari')
    .then((text) => console.log(text));
</script>
```

### UMD in Browser

```html
<script src="https://cdn.jsdelivr.net/npm/lipilekhika"></script>
<script>
  lipilekhika.transliterate('na jAyatE mriyatE vA', 'Normal', 'Devanagari')
    .then((text) => console.log(text));
</script>
```
